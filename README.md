# site1-leyu-guide

本项目用于归档和发布若干独立的 HTML 页面。这些页面可独立运行，不依赖特定域名或网站，主要作为静态内容存档与展示用途。

## 项目简介

本仓库存储了一系列独立的 HTML 文件，内容涵盖信息展示、工具页面或说明文档。所有页面均为纯前端实现，无需后端服务，可直接在浏览器中打开使用。

## 目录说明

```
/
├── index.html       # 项目入口页面（可选）
├── pages/           # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/          # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md        # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面共用的样式、脚本或图片资源。

## 页面归档说明

- 所有页面均为自包含的 HTML 文件，可离线查看。
- 页面之间无强制关联，可单独访问。
- 如需引用公共资源，请确保路径正确（相对于仓库根目录或使用绝对路径）。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下任意 .html 文件，或访问 GitHub Pages 部署的在线版本。
3. 也可自行部署至任意静态服务器。

## 维护说明

- 新增页面时，请在 `pages/` 目录下创建新的 HTML 文件，并遵循已有的文件命名规范。
- 如有公共资源更新，请同步修改 `assets/` 下对应文件。
- 建议定期检查页面兼容性，确保主流浏览器均可正常显示。

## 许可

本项目仅用于内容归档与展示，不涉及商业用途。具体使用请遵守相关法律法规。
